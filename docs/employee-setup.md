# 카페 원고 생성기 — 직원용 설치 가이드

이 프로그램은 Claude Max 구독을 통해 카페 바이럴 원고를 무한히 생성합니다.
사용하려면 **본인의 Claude Max 구독** 이 필요해요.

---

## ✅ 1단계 — Node.js 설치 (이미 있으면 건너뛰기)

1. https://nodejs.org/ko 접속
2. **LTS** (왼쪽, 권장) 다운로드 → 더블클릭 설치
3. 설치 완료 후 PowerShell 또는 cmd 창 열어서 확인:
   ```
   node --version
   npm --version
   ```
   둘 다 버전 번호가 나오면 OK.

## ✅ 2단계 — Claude Code 설치

PowerShell 에서:
```
npm i -g @anthropic-ai/claude-code
```

설치 후:
```
claude login
```
브라우저가 열려요. **본인의 Claude Max 계정** 으로 로그인하세요.

테스트:
```
echo "안녕" | claude -p
```
인사말 답이 돌아오면 준비 완료.

## ✅ 3단계 — 카페 원고 생성기 설치

1. `Cafe Writer Setup 0.1.0.exe` 더블클릭
2. "Windows 가 PC 보호" 경고 뜨면 → **추가 정보** → **실행** (서명 안 된 앱이라 뜨는 거. 안전합니다)
3. 설치 위치 선택 → 설치
4. 바탕화면에 **Cafe Writer** 아이콘 생김

## ✅ 4단계 — 실행

바탕화면 아이콘 더블클릭. 첫 실행 시:
- "Claude Code 가 설치돼있지 않아요" 경고가 뜨면 → 2단계 다시 확인
- 정상이면 잠시 후 창이 열림

창 안에서:
1. `psw1860@naver.com` 같은 관리자에게 가입 승인 요청
2. 가입 → 로그인
3. 제품 템플릿 선택 → 키워드 입력 → 생성

---

## 🔧 자주 묻는 문제

**창이 안 뜨고 바로 종료됨**
- PowerShell 에서 `claude --version` 이 잘 나오는지 확인
- 안 나오면 PATH 에 npm global bin 이 안 들어가있음. Node.js 재설치 추천

**원고 생성이 "Claude Code CLI 응답 시간 초과" 에러**
- 인터넷 연결 확인
- `claude login` 다시 (만료됐을 수 있음)
- Max 일일/주간 한도 초과 가능성 (Claude.ai 가서 확인)

**원고가 영어로 나옴**
- 지침 칸에 한국어 명령이 잘 들어있는지 확인
- 한국어 키워드 명시 (예: "한국어로 작성")

**프로그램 종료**
- 창 X 버튼 → 백그라운드 서버도 자동 종료

## 📦 업데이트

새 버전이 배포되면 똑같이 설치 파일 더블클릭 → 덮어쓰기 설치.
저장된 데이터(템플릿 등) 는 Supabase 에 있어서 사라지지 않음.
